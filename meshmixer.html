01
Introduction to Cranial Reconstruction
Overview · Software Setup · DICOM Basics

Before modeling, understand the clinical context — why cranioplasty is needed, what materials are used, and how 3D printing fits into the surgical workflow.

Software Required: 3D Slicer (free), Autodesk Meshmixer (free), Ultimaker Cura (free), any FDM 3D printer.
  1. Install 3D Slicer from slicer.org — use version 5.x or later
  2. Download and install Meshmixer from Autodesk's official site (free)
  3. Understand DICOM format: a stack of 2D CT slices encoding Hounsfield units (HU)
  4. Learn key HU thresholds: Air ≈ −1000, Soft tissue ≈ 40–80, Bone ≈ 400–1000
  5. Review anatomy: calvaria, diploe, inner/outer cortical tables, suture lines
  6. Familiarize with STL format: tessellated triangular mesh, no color/texture data
💡 Tip: Cranial bone HU values typically range 600–1800. Set your lower threshold to ~250 to capture trabecular bone and avoid gaps in the model.
02
CT to STL Workflow
3D Slicer · DICOM Import · Bone Segmentation · STL Export

Convert raw CT DICOM data into a clean, watertight STL skull model suitable for Meshmixer processing.

  1. Open 3D Slicer → File → Add DICOM Data → import the CT folder
  2. Go to Segment Editor module from the module selector
  3. Click Add to create a new segmentation, name it "Skull"
  4. Select the Threshold effect. Set Lower: 250, Upper: 3000
  5. Click Apply — green voxels show bone tissue
  6. Use Island effect → Keep Largest Island to remove small noise
  7. Apply Smoothing (Gaussian, σ = 1.0 mm) to remove stepping artifact
  8. Right-click segmentation → Export to Files → Format: STL → Save
# 3D Slicer Python console alternative: segmentationNode = slicer.mrmlScene.AddNewNodeByClass("vtkMRMLSegmentationNode") slicer.modules.segmentations.logic().ImportLabelmapToSegmentationNode( labelmapNode, segmentationNode )
💡 Tip: For 1.25 mm slice CT data, the STL will have ~500K–2M triangles. Use Meshmixer's "Reduce" tool to bring it to 300K for workable file size without losing clinical accuracy.
03
Defect Assessment
Meshmixer Inspector · Boundary Analysis · Measurements

Systematically analyze the cranial defect to understand its boundaries, orientation, and relationship to adjacent anatomical structures.

  1. Import the skull STL into Meshmixer: File → Open
  2. Go to Analysis → Inspector — click "Auto Repair All" to fix mesh errors
  3. Use Measure tool (Analysis menu) to record defect dimensions in cm
  4. In View, enable "Show Boundaries" to visualize defect edges (highlighted pink)
  5. Use the Select tool with Lasso to manually mark the defect region
  6. Check for irregularities at the defect margin using Smooth Brush (Shift+B)
  7. Document: anteroposterior distance, mediolateral distance, maximum depth
  8. Identify which skull regions are involved: frontal, temporal, parietal, occipital
💡 Tip: Average FTP (fronto-temporal-parietal) defect post-hemicraniectomy: ~12.8 cm × 11.4 cm. Document this before designing — it defines your implant's outer boundary.
04
Mirroring Technique
Symmetry Selection · Contralateral Mirror · Boolean Operations

Leverage natural bilateral skull symmetry to reconstruct missing anatomy by mirroring the healthy contralateral hemisphere.

  1. With skull open in Meshmixer, switch to Select tool
  2. Enable Symmetry checkbox in the Select options panel
  3. Use Lasso or Brush to select the outer boundary of the skull defect
  4. With Symmetry ON, the mirror selection automatically appears on the opposite side
  5. Disable Symmetry for the defect side; keep only the healthy mirror selection active
  6. Click Edit → Separate to isolate the mirrored healthy tissue as a new object
  7. With the separated piece selected: Edit → Mirror → choose sagittal axis (X)
  8. Align the mirrored piece to the defect using Transform tool if needed
💡 Tip: Perfect symmetry is rare in clinical patients. After mirroring, check the plate's fit by overlaying it on the skull and visually inspecting the transition zones. Minor manual sculpting (Sculpt tools) may be needed at the margins.
05
Surface Creation & Smoothing
Laplacian Smoothing · Sculpt Tools · Edge Refinement
  1. Select the mirrored plate mesh → Deform → Smooth (Laplacian, 10 iterations)
  2. Use Sculpt → Smooth brush (S key) to manually smooth rough transition zones
  3. Use Sculpt → Flatten brush at margins to ensure good bone contact
  4. Trim excess plate extending beyond the defect boundary with Edit → Plane Cut
  5. Inspect underside of plate — it should follow the inner table curvature of the skull
  6. Apply Remesh (Analysis → Remesh) for uniform triangle distribution
  7. Run Inspector again: zero holes, zero non-manifold edges before proceeding
💡 Tip: The "Draw" sculpt brush (D) is excellent for filling small surface depressions. Use a low strength (10–20%) and build up gradually to avoid over-correction.
06
Thickness Generation
Extrude · Boolean Difference · 1mm Offset
  1. With the plate surface selected: Edit → Extrude → Direction: Normal, Distance: 4 mm
  2. This creates a solid PMMA plate of 4 mm nominal thickness (3–5 mm is clinically standard)
  3. To ensure clearance: select the original skull mesh → Edit → Offset → 1 mm outward
  4. Perform Edit → Boolean Difference: skull_offset − plate → removes interference
  5. The result is a plate that sits 1 mm proud of the skull surface, preventing pressure on dura
  6. Add drain holes (~4 mm diameter) randomly across the plate using Boolean with cylinder primitives
  7. Verify minimum wall thickness ≥ 3 mm everywhere using Analysis → Thickness
07
Screw Hole Design
Fixation Points · Titanium Mini-Plate Holes · Boolean Subtraction
  1. Place at least 4 fixation points at the plate periphery, spaced ≥ 15 mm apart
  2. Add cylinder primitive (Meshmixer Primitives) → radius: 0.75 mm, height: 8 mm
  3. Position cylinders perpendicular to the plate surface at each fixation point
  4. Select all cylinders + plate → Edit → Boolean Difference
  5. This creates 1.5 mm diameter through-holes for standard neurosurgical titanium screws
  6. Add countersink rings (thin disc) at each hole for screw head seating if desired
  7. Label fixation point locations in your documentation for the surgical team
💡 Tip: Standard titanium mini-plate screws are 1.5 mm or 2.0 mm diameter. Design holes 0.2–0.3 mm larger to account for FDM printing tolerances and PMMA cast shrinkage (typically 0.3–0.5%).
08
Negative Mold Design
Two-Part Mold · Split Plane · PLA Printing
  1. Duplicate the final plate model (Ctrl+D)
  2. Create a box primitive larger than the plate on all sides by ~10 mm
  3. Perform Boolean Difference: box − plate = negative mold cavity
  4. Use Plane Cut at the plate's widest cross-section to split mold into top and bottom halves
  5. Add alignment pins (4 mm sphere → 2 mm offset) to ensure precise mold registration
  6. Add PMMA pour spout (cone cutout) at the highest point of the mold cavity
  7. Add vent holes (2 mm) at extremities to allow air escape during pouring
  8. Export each mold half as a separate STL for printing
💡 Tip: Print molds with PLA at 0.2 mm layer height, 30% infill, 2 perimeters. STERRAD™ sterilization is compatible with PLA (unlike autoclave which melts it). Cover mold interior with OpSite® drape before casting to prevent adhesion.
09
STL Export & Slicing
Cura Settings · Print Parameters · File Management
  1. In Meshmixer: File → Export → STL Binary (smaller file size)
  2. Open Cura → Add Printer (Creality CR-5 Pro or your FDM printer model)
  3. Import STL → orient flat face down on build plate
  4. Settings: Layer Height: 0.2 mm | Infill: 30% (Gyroid) | Walls: 3 | Speed: 60 mm/s
  5. Enable supports: Tree supports, 45° overhang threshold
  6. Material: PLA, 210°C nozzle, 60°C bed
  7. Estimated print time: 4–8 h for a full FTP plate mold
  8. Slice and preview each layer for artifacts before sending to printer

10
Final Validation
Mesh Integrity · Fit Check · Clinical Sign-off
  1. Run Analysis → Inspector on all final STL files — zero errors required
  2. Use Analysis → Thickness map — confirm minimum 3 mm wall thickness everywhere
  3. Check Overhangs in Cura preview — add supports if angles exceed 45°
  4. Print a physical model of the skull defect (same STL, same scale) for fit testing
  5. After printing the mold, cast a test PMMA piece and physically verify fit on the skull model
  6. Obtain neurosurgeon sign-off on fit, symmetry, and fixation hole positions
  7. Document: implant dimensions (cm), weight (g), material, print date, sterilization date
  8. Archive: all STL files, Meshmixer project, Cura file, surgical photos in patient record
✅ Completed! Your cranial plate design is validated and ready for sterilization and clinical use. See the Case Studies for real-world examples.
